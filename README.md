
# Brainfuck

------------------

A brainfuck interpreter

## Building

```
cargo build
```

## Installing

```
cargo install br4infuck
```

## Usage

```sh
cargo run /path/to/brainfuck_code.bf
```

if you're on mac, you can compile the brainfuck to x86_64 
```
./compiler.sh /path/to/brainfuck_code.bf
```
